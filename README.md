## 🌐 رفع الموقع على الاستضافة (cPanel / Hostinger / Namecheap)

### الطريقة الأسرع — عبر File Manager:

1. **سجّل دخول** على cPanel الخاص بالاستضافة
2. افتح **File Manager**
3. اذهب لمجلد `public_html`
4. **احذف** أي ملفات موجودة (مثل `index.php` الافتراضي)
5. اضغط **Upload** ورفع **كل محتويات هذا المجلد**
6. تأكد إن `index.html` موجود مباشرة في `public_html`

✅ الموقع هيشتغل فوراً على الدومين بتاعك!

---

### رفع عبر FTP (FileZilla):

```
Host:     ftp.yourdomain.com
Username: your_cpanel_user
Password: your_cpanel_password
Port:     21
```

ارفع كل الملفات لـ `/public_html/`

1. اذهب لـ [netlify.com](https://netlify.com)
2. سجّل حساب مجاني
3. اسحب مجلد `ebrahim-portfolio` بالكامل وألقيه في الموقع
4. الموقع هيطلع لك رابط مباشر زي: `https://ebrahim-sayed.netlify.app`
5. ممكن تربط دومين خاص بك مجاناً!
